# Knowledge-distillation

Knowledge distillation is a simple but powerful idea. The main process is to train a smaller model by letting it learn from a bigger and stronger model.
The bigger model is called the teacher and the smaller model is called the student. Instead of learning only from the original dataset, the student can also learn from the teacher's answers, logits, features, explanations, reasoning traces or generated outputs.
I created a beginner friendly PDF on Knowledge Distillation where I collected major KD types, important papers, GitHub repositories, model examples and short summaries.
The goal is simple, to help students, researchers and AI engineers get a quick overview of the KD landscape without getting lost in hundreds of papers.
I will keep updating this resource as I find more useful papers and repos.
